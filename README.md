# E-commerce Website

A full-stack e-commerce platform built with modern web technologies.

## Features

- Product catalog
- Shopping cart
- User authentication
- Order processing
- Responsive design

## Tech Stack

- Frontend: React.js
- Backend: Node.js
- CSS: Tailwind CSS

## Setup

1. Clone the repository
2. Install dependencies:
   ```
   npm install
   ```
3. Start the development server:
   ```
   npm run dev
   ```

## Project Structure

- `/src` - Main application code
- `/public` - Static assets
- `/client` - Frontend code
